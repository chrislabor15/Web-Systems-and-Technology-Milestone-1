About This Project

This “Errands Tracker” is a beginner-friendly, offline web application built using HTML, CSS, and JavaScript.

It includes:

  • Offline login (5 accounts)
  • Calendar system for organizing daily tasks
  • Interactive map for planning errands
  • Dark mode + modern UI
  • Memories page with map pins

This is part of a class project demonstrating multi-page navigation, localStorage usage, and simple data visualization using Leaflet maps.
